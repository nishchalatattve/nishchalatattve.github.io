# `curl`

## Download a file with curl

```console 
$ curl -fsSL <url> -o <output_filename>
```

!!! note "Flags"
    | Flags | Descriptions |
    | ---| ---|
    | `f` | Prevents outputting error page |
    | `s` + `S` | Mute progress bar but show error message |
    | `L` | Location |
    | `o` | Output file name |
