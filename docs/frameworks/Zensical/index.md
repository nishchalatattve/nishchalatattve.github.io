# Zensical

[Zensical] is a static site generator.
It generates static HTTML files from Markdown files.

  [Zensical]: https://zensical.org/

## Installation
```bash
uv init
uv add --dev zensical
```

## Upgrade
```commandline
uv lock --upgrade-package zensical==<version>
```
