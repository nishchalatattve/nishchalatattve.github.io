# uv

## [Installation]

  [Installation]: https://docs.astral.sh/uv/getting-started/installation/


## Initialise a project
```console
$ uv init
```

## Sync an existing projcet
```console
$ uv sync
```

## Upgrade a package
Upgrade `httpx` to "v0.1.0"
```console
$ uv add "httpx>=0.1.0" --upgrade-package httpx
```
