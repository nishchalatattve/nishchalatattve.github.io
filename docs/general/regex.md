# Regular expressions

## Anchors
- `^`: start of string
- `$`: end of string

## Character classes
- `\s`: whitespace character
- `\S`: non-whitespace character

## Quantifiers
- `*`: 0 or more times
- `+`: 1 or more times
- `?`: 0 or 1 times
- `{n}`: exactly n times
- `{n,}`: at least n times
- `{n,m}`: between n and m times

## Groups
- `(<exp>)`: indexed group
