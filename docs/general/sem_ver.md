# Semantic Versioning (SemVer)
[SemVer] is used to indicate the changes in public APIs.
In crude terms, the increase in the _major_ version indicates breaking changes in 
API, the increase in the _minor_ version indicates new features (backward 
compatible), and the increase in the _patch_ version indicates bug fixes (backward 
compatible).
