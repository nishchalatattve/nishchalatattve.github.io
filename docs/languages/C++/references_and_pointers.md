# References and pointers
Pointers are the memory address of an object.
```c++
int number = 4;

int &reference1 =  number;
std::cout << reference1 << std::endl;    // Output: 4

int *address = &number;
std::cout << address << std::endl;  // 0x16d06ee3c
```
In this example, `address` is a pointer. `*address` is an operation called pointer 
deferencing, it returs the value stored in the pointer.
